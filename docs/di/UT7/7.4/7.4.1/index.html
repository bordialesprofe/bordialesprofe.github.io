7.4.1 Widgets básicos

Puedes ampliar esta información

La documentación oficial de Flutter ofrece un catálogo completo de widgets básicos, con ejemplos y explicaciones detalladas.
Puedes consultarlo aquí: https://docs.flutter.dev/ui/widgets

1️⃣ Text​

El widget Text se utiliza para mostrar texto en pantalla.
Permite personalizar su estilo mediante propiedades como el tamaño, el peso, el color o la alineación.
Es uno de los widgets más utilizados en Flutter.

  • Ejemplo
  • Resultado
Text(
  'Hola Flutter',
  style: TextStyle(
    fontSize: 20,
    fontWeight: FontWeight.bold,
  ),
)

2️⃣ Icon​

El widget Icon permite mostrar iconos del conjunto estándar de Material Design.
Se utiliza con frecuencia para complementar textos, destacar acciones o crear botones visuales.
Sus propiedades más habituales son:

  • icon → el icono que quieres mostrar (ejemplo: Icons.star)
  • size → tamaño del icono
  • color → color del icono
  • Ejemplo
  • Resultado
Icon(
  Icons.star,
  size: 40,
  color: Colors.orange,
)

¿Dónde consultar todos los iconos disponibles?

Puedes ver todos los iconos estándar de Material Design en:

  • Catálogo oficial: https://fonts.google.com/icons
  • Documentación de Flutter: https://api.flutter.dev/flutter/material/Icons-class.html

Desde ahí podrás buscar iconos por nombre y utilizar su referencia en Flutter mediante Icons.nombreDelIcono.

3️⃣ Image​

El widget Image permite mostrar imágenes en la interfaz.
Flutter admite imágenes desde distintas fuentes:

  • Image.network → carga imágenes desde una URL
  • Image.asset → carga imágenes locales incluidas en el proyecto
  • Image.file → carga imágenes desde el sistema de archivos
  • Image.memory → carga imágenes desde bytes

En este apartado utilizaremos la opción más común para los primeros ejemplos: Image.network.

Recuerda
  • width y height determinan el tamaño de la imagen.
  • fit: BoxFit.cover ajusta la imagen al contenedor sin deformarla.
  • En Flutter Web, las imágenes de internet pueden tardar un momento en cargarse.
  • Ejemplo
  • Resultado
Image.network(
  'https://picsum.photos/200',
  width: 200,
  height: 200,
  fit: BoxFit.cover,
)

En este ejemplo

El constructor Image.network no puede usar const, porque carga una imagen desde internet en tiempo de ejecución.
const solo puede utilizarse cuando todos los valores son completamente constantes y conocidos en tiempo de compilación, como en const Text('Hola') o const Icon(Icons.star).

Si ves que VS Code te pide quitar const, es porque el widget requiere información dinámica.

4️⃣ Container​

El widget Container es uno de los widgets más versátiles de Flutter.
Permite agregar márgenes, relleno, tamaño, bordes, colores y decoraciones, por lo que se usa constantemente para estructurar y embellecer la interfaz.

Las propiedades más comunes son:

  • padding → espacio interno
  • margin → espacio externo
  • width y height → tamaño
  • decoration → bordes, colores, esquinas redondeadas
  • Ejemplo
  • Resultado
Container(
  padding: EdgeInsets.all(16),
  margin: EdgeInsets.symmetric(vertical: 10),
  color: Colors.grey[300],
  child: Text('Ejemplo de Container'),
)

5️⃣ SizedBox​

El widget SizedBox se utiliza para definir espacios vacíos o para forzar un tamaño específico en un widget hijo.
Es una forma sencilla y eficiente de controlar el espaciado en la interfaz.

Usos comunes:

  • Crear espacios verticales (height)
  • Crear espacios horizontales (width)
  • Fijar un tamaño a un widget envolviéndolo en un SizedBox
  • Ejemplo
  • Resultado
Column(
  mainAxisSize: MainAxisSize.min,
  children: [
    Text('Arriba'),
    SizedBox(height: 30), // espacio visible
    Text('Abajo'),
  ],
)

Consejo

Usa SizedBox para generar espacio entre widgets en lugar de insertar Containers vacíos.
Es más limpio y más eficiente.

6️⃣ Actividad: Construcción de una interfaz básica con widgets básicos​

  • Enunciado
  • Código
  • Resultado

En esta actividad construirás una interfaz sencilla utilizando los widgets básicos de Flutter. Tu objetivo es crear una pantalla que muestre varios elementos visuales organizados dentro de una columna y centrados verticalmente. Para ello deberás utilizar los siguientes widgets:

  • Text
  • Icon
  • Image
  • SizedBox (para añadir espacios)
  • Container (para añadir diseño y estructura)

Lo que debe incluir tu pantalla

  1. Un texto principal con estilo (tamaño y peso de fuente).
  2. Un icono visible y centrado.
  3. Una imagen obtenida desde internet.
  4. Separaciones verticales entre los elementos usando SizedBox.
  5. Un Container con color de fondo, padding, borde y esquinas redondeadas.

Todo organizado dentro de un widget Column centrado en pantalla.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(home: HomePage());
  }
}

class HomePage extends StatelessWidget {
  const HomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text('Widgets Básicos')),
      body: Center(
        child: Column(
          mainAxisSize: MainAxisSize.min,
          children: [
            Text(
              'Bienvenido a Flutter',
              style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold),
            ),

            SizedBox(height: 20),

            Icon(Icons.star, size: 40, color: Colors.orange),

            SizedBox(height: 20),

            Image.network(
              'https://picsum.photos/200',
              width: 200,
              height: 200,
              fit: BoxFit.cover,
            ),

            SizedBox(height: 20),

            Container(
              padding: EdgeInsets.all(16),
              margin: EdgeInsets.symmetric(horizontal: 16),
              decoration: BoxDecoration(
                color: Colors.blue[50],
                borderRadius: BorderRadius.circular(12),
                border: Border.all(color: Colors.blue),
              ),
              child: Text(
                'Este es un ejemplo de Container con estilo.',
                textAlign: TextAlign.center,
              ),
            ),
          ],
        ),
      ),
    );
  }
}

Buenas prácticas iniciales con widgets básicos
  • Usa SizedBox para añadir espacios entre widgets cuando organices tu diseño en vertical u horizontal (lo veremos con más detalle en el apartado de widgets estructurales).
  • Emplea Container cuando necesites añadir color, márgenes o mejorar la estructura visual del diseño.
  • Aplica estilo a los textos mediante TextStyle para mejorar la jerarquía tipográfica y la legibilidad.
  • Evita anidar contenedores innecesarios: un Container vacío dentro de otro rara vez aporta valor.
