7.8 Flutter Inspector

Flutter DevTools es un conjunto de herramientas de depuración y análisis integrado en el SDK de Flutter y Dart. Estas herramientas permiten observar el comportamiento interno de una aplicación mientras se encuentra en ejecución.

Dentro de DevTools, el Flutter Inspector se utiliza para analizar visualmente la interfaz de usuario y explorar la estructura de widgets que la componen.

A medida que las interfaces crecen en complejidad, no siempre resulta evidente identificar qué widget genera determinados espacios, alineaciones incorrectas o desbordamientos. El Inspector permite visualizar la jerarquía de widgets, el tamaño y las restricciones que recibe cada uno, así como la forma en que se distribuye el espacio dentro de estructuras como Column, Row o ListView.

En este apartado se introduce el uso del Flutter Inspector como herramienta para la inspección del árbol de widgets y el análisis visual de la estructura de la interfaz en aplicaciones Flutter.

1️⃣ Acceso a Flutter Inspector desde VS Code​

Para utilizar Flutter Inspector es necesario ejecutar la aplicación en modo depuración.

El acceso se realiza siguiendo estos pasos:

  1. Abrir el proyecto Flutter en Visual Studio Code.
  2. Iniciar la aplicación en modo depuración mediante el menú Ejecutar → Iniciar depuración o pulsando la tecla F5.
  3. Abrir la paleta de comandos (F1).
  4. Ejecutar el comando: Dart: Open DevTools Una vez iniciado, DevTools se abre mostrando las distintas herramientas disponibles, entre ellas el Flutter Inspector.

Acceso a Flutter Inspector desde VS Code

2️⃣ DevTools integrado y DevTools en navegador​

Flutter DevTools puede mostrarse de dos formas distintas:

  • Integrado en VS Code, como un panel dentro del editor.
  • En el navegador, en una pestaña independiente.
  • DevTools integrado
  • DevTools en navegador
EN EL NAVEGADOR

Cuando DevTools se abre en el navegador, se muestra la interfaz completa del Inspector, que incluye herramientas visuales adicionales para el análisis del diseño y la estructura de la interfaz. Este modo permite una exploración más detallada del árbol de widgets y de sus propiedades.

3️⃣ Inspección del árbol de widgets​

El Widget Tree representa la estructura jerárquica real de la interfaz.

En el árbol de widgets pueden identificarse, entre otros:

  • Widgets de estructura general como MaterialApp y Scaffold.
  • Widgets de organización como Column, Row, Padding o Container.
  • Widgets de contenido como Text, Image o Icon.

Cada widget puede seleccionarse para mostrar información asociada, como su tamaño, alineación o espaciado, lo que permite comprender cómo se distribuyen los elementos en pantalla.

Inspección del árbol de widgets

4️⃣ Inspección visual de propiedades​

Al seleccionar un widget en el árbol, Flutter Inspector muestra información relevante sobre dicho widget, entre la que se incluye:

  • Dimensiones (alto y ancho).
  • Padding aplicado.
  • Alineación en los ejes principal y secundario.
  • Dirección de distribución en widgets como Row y Column.

Esta información se presenta de forma visual, facilitando la comprensión de la disposición de los elementos sin necesidad de modificar el código.

5️⃣ Herramientas principales del Inspector​

Flutter Inspector incluye varias herramientas visuales que permiten analizar el diseño de la interfaz:

Herramientas Flutter Inspector

  • Select Widget Mode Permite seleccionar un widget directamente desde la aplicación en ejecución y localizarlo en el árbol.

  • Show Implementation Widgets Muestra los widgets internos utilizados por Flutter para implementar otros widgets.

  • Show Guidelines Superpone guías visuales para facilitar la detección de problemas de alineación.

  • Show Baselines Muestra las líneas base del texto para comprobar la alineación tipográfica.

  • Highlight Repaints Resalta las zonas de la interfaz que se repintan durante la ejecución.

  • Highlight Oversized Images Identifica imágenes cuyo tamaño es mayor del necesario y que pueden afectar al consumo de memoria.

Estas herramientas permiten realizar una depuración visual del diseño de la interfaz.

6️⃣ Otras funcionalidades de DevTools​

Además del Inspector, DevTools incluye herramientas orientadas al análisis de:

  • Rendimiento de la aplicación.
  • Uso de CPU.
  • Memoria.
  • Red y depuración a nivel de código.

Estas funcionalidades se abordarán en unidades posteriores, cuando se introduzca el manejo de estado y la lógica de las aplicaciones Flutter.
