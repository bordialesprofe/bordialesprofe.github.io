8.3.1 Identificación de componentes reutilizables

El siguiente paso para separar el código por responsabilidad consiste en extraer los componentes visuales reutilizables fuera de la pantalla. De esta forma, la pantalla se centra en mostrar la interfaz y gestionar la interacción, y los componentes se definen en archivos independientes dentro de widgets/.

En el Salón de los Héroes, salon_heroes_screen.dart contiene:

  • la pantalla principal (Scaffold, AppBar, body…)
  • el estado y la lógica de interacción (selección, reset, SnackBar…)
  • una función heroCard(...) que construye la tarjeta de un héroe

Esto funciona, pero hace que la pantalla mezcle responsabilidades. La tarjeta es un componente visual reutilizable y debe estar separada.

A continuación se detallan los pasos para separar el componente HeroCard.

1️⃣ Crear el archivo del componente​

Dentro de lib/widgets/, se crea un nuevo archivo para la tarjeta de héroe (hero_card.dart). Este archivo contendrá:

  • el widget HeroCard como clase StatelessWidget
  • su estructura visual completa (Container, Row, Text, Image…)
  • sus parámetros de entrada (imagePath, nombre, clase, descripcion, seleccionado)
Convención de nombres para componentes

En Flutter es habitual nombrar los widgets reutilizables con nombres descriptivos en UpperCamelCase. El archivo se escribe en snake_case.

Ejemplo:

  • Clase: HeroCard
  • Archivo: hero_card.dart

2️⃣ Mover el código de la tarjeta​

Desde salon_heroes_screen.dart se mueve:

  • el contenido completo de la función heroCard(...)
  • la estructura de UI de la tarjeta
  • los estilos propios del componente

En este momento no se cambia el contenido visual del componente, solo su ubicación y formato, pasando de función a clase.

3️⃣ Actualizar la pantalla para usar el componente​

Una vez creado el widget, la pantalla deja de llamar a heroCard(...) y pasa a utilizar HeroCard(...) en el listado.

En este punto:

  • la pantalla sigue controlando el estado (heroeSeleccionado)
  • la tarjeta solo recibe datos y muestra la interfaz

La lógica de interacción se ajustará en el siguiente apartado mediante callbacks

lib/widgets/hero_card.dart
import 'package:flutter/material.dart';

class HeroCard extends StatelessWidget {
  final String imagePath;
  final String nombre;
  final String clase;
  final String descripcion;
  final bool seleccionado;

  const HeroCard({
    super.key,
    required this.imagePath,
    required this.nombre,
    required this.clase,
    required this.descripcion,
    required this.seleccionado,
  });

  @override
  Widget build(BuildContext context) {
    return Container(
      padding: const EdgeInsets.all(12),
      decoration: BoxDecoration(
        color: seleccionado ? const Color(0xFFDFF2DF) : Colors.white,
        borderRadius: BorderRadius.circular(12),
        border: Border.all(
          color: const Color(0xC1035B02),
          width: seleccionado ? 4 : 2,
        ),
      ),
      child: Row(
        children: [
          Image.asset(imagePath, width: 64, height: 64),
          const SizedBox(width: 12),
          Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              Text(
                nombre,
                style: const TextStyle(
                  fontSize: 18,
                  fontWeight: FontWeight.bold,
                ),
              ),
              Text(
                clase,
                style: const TextStyle(fontSize: 14, color: Colors.grey),
              ),
            ],
          ),
        ],
      ),
    );
  }
}
lib/screens/salon_heroes_screen.dart
import '../data/heroes_data.dart';
import '../models/heroes_model.dart';
import '../widgets/hero_card.dart';

class SalonHeroesScreen extends StatefulWidget {
  const SalonHeroesScreen({super.key});

  @override
  State<SalonHeroesScreen> createState() => _SalonHeroesScreenState();
}

class _SalonHeroesScreenState extends State<SalonHeroesScreen> {
  String heroeSeleccionado = '';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text(
          'Salón de héroes',
          style: TextStyle(fontWeight: FontWeight.bold),
        ),
        centerTitle: true,
        backgroundColor: const Color(0xC1035B02),
        foregroundColor: const Color(0xC1FFFFFF),
        actions: [
          IconButton(
            icon: const Icon(Icons.refresh),
            tooltip: 'Limpiar selección',
            onPressed: () {
              setState(() {
                heroeSeleccionado = '';
              });
            },
          ),
        ],
      ),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: ListView.separated(
          itemCount: heroes.length,
          separatorBuilder: (context, index) => const SizedBox(height: 12),
          itemBuilder: (context, index) {
            final HeroeModel heroe = heroes[index];

            return InkWell(
              onTap: () {
                setState(() {
                  heroeSeleccionado = heroe.nombre;
                });

                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(content: Text(heroe.descripcion)),
                );
              },
              child: HeroCard(
                imagePath: heroe.imagePath,
                nombre: heroe.nombre,
                clase: heroe.clase,
                descripcion: heroe.descripcion,
                seleccionado: heroeSeleccionado == heroe.nombre,
              ),
            );
          },
        ),
      ),
    );
  }
}
