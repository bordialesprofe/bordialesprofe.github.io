5.8.1 Float

El uso de floats fue una de las primeras técnicas empleadas en CSS para crear diseños multicolumna. Aunque hoy en día se prefiere Flexbox o Grid, comprender los floats sigue siendo útil para mantener o interpretar código heredado.

Cuando un elemento se flota, se saca parcialmente del flujo normal y se alinea hacia la izquierda o derecha dentro de su contenedor. El texto y otros elementos que no estén flotados se ajustan automáticamente alrededor de él.

1️⃣ Concepto básico​

El flujo normal de los elemtnos de bloque es posicionarse uno debajo de otro. Mediante la propiedad float podemos modificar este flujo vertical y transformar algunos elementos de bloque asignándoles una horientación horizontal.

La propiedad float permite mover un elemento hacia un lado del contenedor (left o right) y hacer que los demás contenidos fluyan a su alrededor. Fue pensada originalmente para alinear imágenes dentro de un párrafo de texto.

ValorDescripción
leftFlota el elemento a la izquierda.
rightFlota el elemento a la derecha.
noneEl elemento sigue el flujo normal (valor por defecto).
Comportamiento por defecto

El comportamiento por defecto de un bloque es emplear todo el ancho de su contenedor afectando al flujo vertical de la página.

.rojo {
  width: 200px;
  eight: 300px
  background-color: #f09a9d;
}
en este ejemplo

El bloque de color rojo ocupa todo el ancho de la página aunque su anchura total sea solo de 200 píxeles.

Comportamiento con float

Si aplicamos la prropiedad float al bloque, este dejará de afectar al flujo vertical de la página flotando en su contenedor.

.rojo {
  float: left;
  width: 200px;
  eight: 300px
  background-color: #f09a9d;
}
en este ejemplo

El bloque flota en su contenedor dejando de ocupar el flujo vertical de la página.

2️⃣ Propiedad clear​

La propiedad clear indica que un elemento no puede situarse junto a un float en un lado concreto. Se usa comúnmente para que un bloque comience debajo de los elementos flotados.

Con la propiedad clear le decimos a un bloque que ignore los floats que aparecen delante de él. Es como forzar que un cuadro vuelva al flujo vertical predeterminado de la página.

ValorDescripción
leftNo permite elementos flotados a la izquierda.
rightNo permite elementos flotados a la derecha.
bothNo permite floats a ninguno de los lados.

Como hemos visto anteriormente, los elementos flotantes no afectan al flujo normal de bloques de una página.

Aplicando la propiedad clear le decimos a un bloque que ignore los floats que aparecen delante de él. Es como forzar que un cuadro vuelva al flujo vertical predeterminado de la página.

aplicamos la propiedad clear al footer
.footer {
  clear: both;  
  height: 200px;
  background-color: #D6E9FE;
}

El resultado es que el pie vuelve a tener en cuenta a los bloques float para ocupar el flujo vertical de la página.

3️⃣ Uso actual de float​

Aunque, actualmente, Flex haya sustituido a los Floats, estos siguen siendo útiles si quieres mostrar contenido alrededor de una imagen.

Estilo para imagen con texto alrededor
    .imagen_articulo {
      float: left;
      width: 300px;
      height: 200px;
      margin-right: 20px;
      margin-bottom: 20px;
    }
overflow: hidden;

Puede que en alguna ocasión queramos que el texto no rodee la imagen sino que se quede a un lado. Para esto puede sernos util modificar la propiedad overflor del párrafo y darle el valor hidden

<div class='column'>
  <div class='avatar'></div>
  <h3 class='username'>Bob Smith</h3>
  <p class='comment'>Aptent vel egestas vestibulum aliquam ullamcorper volutpat
  ullamcorper pharetra hac posuere a rhoncus purus molestie torquent. Scelerisque
  purus cursus dictum ornare a phasellus. A augue venenatis adipiscing.</p>
</div>
.avatar {
  float: left;
  width: 60px;
  height: 60px;
  margin: 25px;
  border-radius: 40px;
  background-color: #D6E9FE;
}
.username {
  margin-top: 30px;
}
.comment {
  margin: 10px;
  overflow: hidden; 
}
