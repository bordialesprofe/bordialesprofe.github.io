UT6 - Responsive Web Design

En esta unidad se estudia Responsive Web Design (RWD): el conjunto de técnicas que permiten crear una misma web capaz de adaptarse a diferentes tamaños de pantalla (móvil, tablet y escritorio) manteniendo una buena lectura, navegación y usabilidad.

Hoy en día, el acceso a Internet se realiza desde dispositivos muy distintos y no tiene sentido diseñar pensando en un único tamaño. Por ello, el objetivo no es “hacer la versión móvil”, sino diseñar un interfaz flexible que se reorganiza cuando el espacio disponible cambia.

1️⃣ Ventajas de aplicar Responsive Web Design​

  • Mejor experiencia de usuario: el contenido se ve claro y se usa cómodo en cualquier dispositivo.

  • Mantenimiento más sencillo: una sola web y una sola base de código.

  • Mejor consistencia: el diseño y la información son coherentes entre dispositivos.

2️⃣ Qué veremos en esta unidad​

En esta unidad se aprenderá a:

  • Comprender los principios básicos del diseño responsive: fluidez, breakpoints y estrategias como mobile-first.

  • Aplicar media queries para ajustar la maquetación cuando el diseño lo necesite.

  • Usar container queries para adaptar componentes en función del espacio de su contenedor.

  • Respetar preferencias del usuario que afectan al diseño, como modo oscuro o reducción de movimiento.
