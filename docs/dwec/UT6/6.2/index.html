6.2 Clases en JavaScript

Cuando se necesitan crear muchos objetos con la misma estructura, definirlos uno a uno deja de ser práctico y aumenta el riesgo de errores. Para resolver este problema, JavaScript permite definir clases, que actúan como un modelo o plantilla para crear objetos.

Una clase define:

  • Las propiedades que tendrán los objetos.
  • Los métodos que podrán ejecutar.

A partir de una clase se pueden crear tantos objetos como sea necesario.

1️⃣ Definición de una clase​

Una clase se define utilizando la palabra clave class.

class Usuario {
  constructor(nombre, email) {
    this.nombre = nombre;
    this.email = email;
  }

  saludar() {
    console.log("Hola, soy " + this.nombre);
  }
}

En esta clase:

  • Usuario es el nombre de la clase.
  • constructor es un método especial que se ejecuta al crear el objeto.
  • this hace referencia al objeto que se está creando.
  • saludar() es un método común a todos los usuarios.

2️⃣ Crear objetos a partir de una clase​

Para crear un objeto a partir de una clase se utiliza la palabra clave new.

let usuario1 = new Usuario("Ana", "ana@email.com");
let usuario2 = new Usuario("Carlos", "carlos@email.com");

Aunque se han creado dos objetos distintos:

  • Ambos tienen las mismas propiedades.
  • Ambos pueden ejecutar los mismos métodos.
  • Cada uno mantiene sus propios valores.
usuario1.saludar();
usuario2.saludar();

3️⃣ Ventajas de usar clases​

En una aplicación web real, las clases suelen representar:

  • Usuarios
  • Productos
  • Pedidos
  • Mensajes
  • Tareas

Cada vez que se necesita uno de estos elementos, se crea un nuevo objeto a partir de la clase correspondiente.

4️⃣ Actividad​

  • Ejemplo
  • Código

Primeros clases

  1. Crea una clase llamada Producto.

  2. El constructor debe recibir:

    • nombre
    • precio
    • stock

  3. Añade un método mostrarInfo() que muestre por consola la información del producto.

  4. Crea al menos dos productos distintos y muestra su información.

class Producto {
constructor(nombre, precio, stock) {
  this.nombre = nombre;
  this.precio = precio;
  this.stock = stock;
}

mostrarInfo() {
  console.log(
    "Producto: " + this.nombre +
    " | Precio: " + this.precio + " €" +
    " | Stock: " + this.stock
  );
}
}

// Crear productos
let producto1 = new Producto("Portátil", 850, 10);
let producto2 = new Producto("Ratón", 25, 50);

// Mostrar información
producto1.mostrarInfo();
producto2.mostrarInfo();
