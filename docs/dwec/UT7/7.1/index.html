7.1 Objetos predefinidos del lenguaje

En JavaScript existen objetos predefinidos, es decir, objetos que ya vienen incorporados en el propio lenguaje y están disponibles desde el inicio. Estos objetos aportan propiedades y métodos ya preparados para resolver tareas habituales sin tener que programarlas desde cero. Por ejemplo, permiten trabajar con texto, números, cálculos matemáticos, fechas o validaciones.

Para entenderlos bien conviene distinguir entre valores primitivos y objetos:

  • Un primitivo es un valor “simple”, como un texto, un número o un booleano. Ejemplos: "hola", 25, true.
  • Un objeto es una estructura que puede tener propiedades y métodos.

En el caso del texto, string (minúscula) es el tipo primitivo, mientras que String (mayúscula) es el objeto/constructor que incluye métodos para trabajar con cadenas. Aunque se use un texto primitivo, JavaScript permite utilizar métodos como si fuese un objeto:

console.log("hola".toUpperCase()); // "HOLA"
console.log("   hola   ".trim());  // "hola"

Esto ocurre porque JavaScript realiza una conversión temporal para poder aplicar los métodos del objeto correspondiente al valor primitivo.

1️⃣ Objeto String​

String permite trabajar con cadenas de texto. Se usa para transformar, buscar, extraer o limpiar texto (muy común en datos introducidos por el usuario).

🔶 Propiedades​

PropiedadDescripciónEjemplo
lengthDevuelve el número de caracteres de la cadena"Hola".length → 4

🔶 Métodos​

MétodoDescripciónEjemplo
toUpperCase()Convierte el texto a mayúsculas"hola".toUpperCase() → "HOLA"
toLowerCase()Convierte el texto a minúsculas"HOLA".toLowerCase() → "hola"
trim()Elimina espacios al inicio y al final" hola ".trim() → "hola"
includes(texto)Comprueba si contiene un texto"Hola mundo".includes("mundo") → true
indexOf(texto)Devuelve la posición o -1 si no aparece"Hola".indexOf("a") → 3
slice(inicio, fin)Extrae una parte de la cadena"JavaScript".slice(0,4) → "Java"
substring(inicio, fin)Extrae una parte (similar a slice)"JavaScript".substring(4,10) → "Script"
replace(a, b)Sustituye la primera coincidencia"Hola mundo".replace("mundo","JS") → "Hola JS"
split(sep)Divide el texto en un array"a,b,c".split(",") → ["a","b","c"]
aviso

En JavaScript se trabaja normalmente con string (primitivo). String (con mayúscula) es el objeto/constructor que aporta los métodos para trabajar con texto.

Aunque se use un string, se pueden usar métodos como toUpperCase() porque JavaScript lo convierte temporalmente en un objeto String.

Crear textos con new String("...") no se recomienda, porque genera un objeto y puede dar resultados inesperados en comparaciones.

const a = "hola";              // string (primitivo)
const b = new String("hola");  // String (objeto) -> no recomendado

console.log(typeof a); // "string"
console.log(typeof b); // "object"

// Los métodos funcionan en ambos:
console.log(a.toUpperCase()); // "HOLA"
console.log(b.toUpperCase()); // "HOLA"

// Diferencia importante:
console.log(a === "hola"); // true
console.log(b === "hola"); // false (b es un objeto)
console.log(b.valueOf() === "hola"); // true (convierte a primitivo)
Recuerda
  • valueOf() devuelve el valor primitivo de un objeto. En la práctica, se usa poco, porque normalmente se trabaja con primitivos y no con wrappers como new String("..."), que no se recomienda usar.
const b = new String("hola");  // objeto
console.log(b.valueOf());      // "hola" (primitivo)
  • typeof devuelve el tipo de un dato en JavaScript. Es útil para comprobar si algo es texto ("string") o número ("number").
console.log(typeof "18"); // "string"
console.log(typeof 18);   // "number"

2️⃣ Objeto Number​

Number permite trabajar con números y, sobre todo, realizar conversiones desde texto a valores numéricos. Es muy común cuando se leen datos de un formulario (que llegan como texto) y se necesita operar con ellos.

🔶 Propiedades​

PropiedadDescripciónEjemplo
NaN“Not a Number”, representa un valor numérico inválidoNumber.NaN
MAX_VALUEMayor número representableNumber.MAX_VALUE
MIN_VALUEMenor número positivo representableNumber.MIN_VALUE
isNaN(valor)Comprueba si un valor es NaNNumber.isNaN(NaN) → true
isFinite(valor)Comprueba si es un número finitoNumber.isFinite(10) → true

🔶 Métodos​

MétodoDescripciónEjemplo
Number(valor)Convierte a número (si se puede)Number("12") → 12
parseInt(texto)Convierte a enteroparseInt("12.9") → 12
parseFloat(texto)Convierte a decimalparseFloat("12.9") → 12.9
toFixed(n)Devuelve el número con n decimales (como texto)(3.1416).toFixed(2) → "3.14"
Nota: los valores de formularios llegan como texto

Aunque el campo sea numérico, lo que se recibe suele ser un string. Para operar, primero se convierte.

const edadTexto = "18";
const edad = Number(edadTexto);

console.log(typeof edadTexto); // "string"
console.log(typeof edad);      // "number"
console.log(edad + 1);         // 19
Nota: NaN y conversiones

Si la conversión falla, el resultado puede ser NaN. Para comprobarlo, se recomienda Number.isNaN(...).

const x = Number("hola"); 
console.log(x); // NaN
console.log(Number.isNaN(x)); // true

3️⃣ Objeto Math​

Math es un objeto que incluye propiedades y métodos matemáticos. Se utiliza para redondeos, cálculos comunes y para generar números aleatorios.

🔶 Propiedades​

Si la propiedad ya existe, se puede cambiar su valor directamente:

PropiedadDescripciónEjemplo
PIValor de πMath.PI → 3.141592...

🔶 Métodos​

MétodoDescripciónEjemplo
random()Número aleatorio entre 0 (incluido) y 1 (excluido)Math.random()
round(n)Redondea al entero más cercanoMath.round(3.6) → 4
floor(n)Redondea hacia abajoMath.floor(3.9) → 3
ceil(n)Redondea hacia arribaMath.ceil(3.1) → 4
max(a, b, ...)Devuelve el mayorMath.max(2, 9, 4) → 9
min(a, b, ...)Devuelve el menorMath.min(2, 9, 4) → 2
Ejemplo
// Número aleatorio (0 <= n < 1)
const n = Math.random();
console.log(n);

// Redondeos
console.log(Math.round(4.5)); // 5
console.log(Math.floor(4.9)); // 4
console.log(Math.ceil(4.1));  // 5
Aleatorio en un rango
  • Para obtener un entero aleatorio entre min y max (incluidos): Math.floor(Math.random() * (max - min + 1)) + min
  • Para obtener un entero aleatorio entre 1 y max: Math.floor(Math.random() * max) + 1
Número aleatorio de 1 a 6 (como un dado)
const dado = Math.floor(Math.random() * 6) + 1;
Explicación paso a paso
  • Math.random() devuelve un número decimal aleatorio entre 0 (incluido) y 1 (excluido).
  • Al multiplicar por 6 (Math.random() * 6) el rango pasa a ser 0 ≤ n < 6.
  • Math.floor(...) elimina los decimales y redondea hacia abajo, dejando valores posibles 0, 1, 2, 3, 4, 5.
  • Al sumar + 1 el rango se desplaza a 1, 2, 3, 4, 5, 6.

4️⃣ Objeto Date​

Date permite trabajar con fechas y horas. Se utiliza para obtener la fecha actual, mostrarla en distintos formatos y calcular o modificar fechas (por ejemplo, sumar días).

🔶 Propiedades​

En Date se trabaja sobre todo con el constructor y métodos

🔶 Métodos​

MétodoDescripciónEjemplo
new Date()Crea un objeto con la fecha y hora actualesnew Date()
getFullYear()Devuelve el añofecha.getFullYear()
getMonth()Devuelve el mes (0–11)fecha.getMonth()
getDate()Devuelve el día del mes (1–31)fecha.getDate()
getDay()Devuelve el día de la semana (0–6)fecha.getDay()
getHours()Devuelve la hora (0–23)fecha.getHours()
getMinutes()Devuelve los minutos (0–59)fecha.getMinutes()
getSeconds()Devuelve los segundos (0–59)fecha.getSeconds()
setDate(n)Cambia el día del mesfecha.setDate(20)
toLocaleDateString()Devuelve la fecha en formato localfecha.toLocaleDateString()
toLocaleTimeString()Devuelve la hora en formato localfecha.toLocaleTimeString()
Importante: meses y días de la semana
  • getMonth() devuelve el mes desde 0 (enero) hasta 11 (diciembre).
  • getDay() devuelve el día de la semana desde 0 (domingo) hasta 6 (sábado).
Ejemplo
const fecha = new Date();

console.log(fecha); // fecha y hora completas
console.log(fecha.getFullYear()); // año
console.log(fecha.getMonth());    // mes (0-11)
console.log(fecha.getDate());     // día del mes

console.log(fecha.toLocaleDateString()); // fecha en formato local
console.log(fecha.toLocaleTimeString()); // hora en formato local
sumar días a una fecha

Para sumar días, se puede usar setDate() sobre el valor actual.

const f = new Date();
f.setDate(f.getDate() + 7); // suma 7 días
console.log(f.toLocaleDateString());

5️⃣ Objeto RegExp​

RegExp permite trabajar con expresiones regulares, que son patrones para comprobar si un texto cumple un formato (por ejemplo, un email) o para buscar coincidencias dentro de una cadena.

🔶 Propiedades​

PropiedadDescripciónEjemplo
flagsDevuelve los modificadores usados/abc/gi.flags → "gi"
sourceDevuelve el patrón sin las barras / //abc/gi.source → "abc"

🔶 Métodos​

MétodoDescripciónEjemplo
test(texto)Devuelve true/false si el texto cumple el patrón/^a/.test("anillo") → true
exec(texto)Devuelve info de la coincidencia (o null)/a./.exec("casa")
Formas de crear una expresión regular
  • Literal: /patron/flags
  • Constructor: new RegExp("patron", "flags")
const r1 = /hola/i; // i = ignora mayúsculas/minúsculas
console.log(r1.test("HOLA")); // true

const r2 = new RegExp("^hola$", "i"); // empieza y termina en "hola"
console.log(r2.test("hola"));  // true
console.log(r2.test("hola!")); // false

🔶 Patrones más usados​

ElementoSignificadoUso típicoEjemplo
iIgnora mayúsculas/minúsculasBúsqueda/validación sin distinguir mayúsculas/hola/i
gBusca todas las coincidenciasEncontrar todas las apariciones/a/g

\SUn carácter NO espacioEvitar espacios/\S+/

[]Uno de los caracteres indicados (clase)Validar conjuntos (vocales, rangos)/[aeiou]/
[^]Cualquier carácter excepto los indicadosProhibir caracteres (solo no dígitos, etc.)/[^0-9]/

``Alternativa (una opción u otra)Validar opciones (extensiones, prefijos)
Importante: probar varios casos

Un patrón mal planteado puede validar cosas que no debería o rechazar cosas válidas. Conviene probar varios casos antes de darlo por bueno.

Ejemplo: validar un email (simple)
const patronEmail = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;

console.log(patronEmail.test("usuario@gmail.com")); // true
console.log(patronEmail.test("usuario@gmail"));     // false
console.log(patronEmail.test("usuario @gmail.com"));// false
trim()

Antes de validar, conviene eliminar espacios con trim().

const email = "  usuario@gmail.com  ";
console.log(patronEmail.test(email.trim())); // true
