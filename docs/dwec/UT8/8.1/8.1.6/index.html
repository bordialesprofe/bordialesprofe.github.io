8.1.6 Creación, inserción y eliminación de elementos

Además de recorrer el DOM, muchas veces se necesita crear elementos nuevos, insertarlos en una zona concreta de la página o eliminarlos cuando ya no se usan. Para ello se utilizan métodos como createElement(), append(), prepend(), insertBefore(), remove() o removeChild().

1️⃣ Crear elementos con createElement()​

Cuando se quiere generar contenido dinámico (tarjetas, filas, mensajes, etiquetas), primero se crea el elemento con document.createElement(). Este método crea el nodo, pero todavía no aparece en la página hasta que se inserta.

Sintaxis
document.createElement("etiqueta")
  • HTML
  • JavaScript
<section id="lista-eventos"></section> 
const tarjeta = document.createElement("article"); 
tarjeta.classList.add("tarjeta"); 
tarjeta.textContent = "Nuevo evento creado";
en este ejemplo

Se crea un elemento <article>, se le añade una clase y se define su texto, pero todavía no se inserta en el DOM.

2️⃣ Insertar al final con append() / appendChild()​

Una vez creado el elemento, se debe insertar dentro de un contenedor.

  • append() lo añade al final del contenedor y permite añadir varios nodos o texto.
  • appendChild() hace algo similar, pero solo admite un nodo.
Sintaxis
contenedor.append(elemento)
  • HTML
  • JavaScript
<section id="lista-eventos"></section>
const tarjeta = document.createElement("article");
tarjeta.classList.add("tarjeta");
tarjeta.textContent = "Reunión de departamento";
lista.append(tarjeta);
en este ejemplo

Se crea una tarjeta y se inserta al final del contenedor #lista-eventos, por lo que ya aparece en la página.

3️⃣ Insertar al inicio con prepend()​

Cuando interesa que lo nuevo aparezca arriba (por ejemplo, el evento más reciente), se usa prepend(), que inserta el elemento como primer hijo del contenedor.

Sintaxis
contenedor.prepend(elemento)
  • HTML
  • JavaScript
<section id="lista-eventos"> 
    <article class="tarjeta">Aviso anterior</article> 
</section>
const lista = document.querySelector("#lista-eventos");
const tarjeta = document.createElement("article");
tarjeta.classList.add("tarjeta");
tarjeta.textContent = "Aviso importante";
lista.prepend(tarjeta);
en este ejemplo

La tarjeta nueva se inserta como primera, quedando por encima del contenido que ya existía.

4️⃣ Insertar antes de otro elemento con insertBefore()​

A veces se necesita insertar un elemento en una posición concreta (no al inicio ni al final). Para ello se puede usar insertBefore(), indicando el elemento nuevo y el elemento de referencia delante del cual se insertará.

Sintaxis
padre.insertBefore(nuevoElemento, elementoReferencia)
  • HTML
  • JavaScript
<section id="lista-eventos"> 
    <article class="tarjeta" id="ref">Evento existente</article> 
</section>
const lista = document.querySelector("#lista-eventos"); 
const referencia = document.querySelector("#ref");
const tarjeta = document.createElement("article");
tarjeta.classList.add("tarjeta");
tarjeta.textContent = "Evento insertado antes";
lista.insertBefore(tarjeta, referencia);
en este ejemplo

Se inserta una nueva tarjeta justo antes de la tarjeta con id ref, manteniendo el orden deseado.

5️⃣ Eliminar elementos con remove()​

Para borrar un elemento del DOM se puede usar remove() directamente sobre el propio elemento. Es la opción más simple cuando ya se tiene una referencia al nodo.

Sintaxis
elemento.remove()
  • HTML
  • JavaScript
<article class="tarjeta"> 
    <button class="btn-eliminar">Eliminar</button> 
</article> 
const boton = document.querySelector(".btn-eliminar"); 
const tarjeta = boton.closest(".tarjeta");
tarjeta.remove();
en este ejemplo

Se localiza la tarjeta completa desde el botón y se elimina del DOM, desapareciendo de la página.

6️⃣ Eliminar desde el padre con removeChild()​

En algunos casos se elimina un nodo desde su padre usando removeChild(). Es un método más antiguo que sigue apareciendo en ejemplos y documentación.

Sintaxis
padre.removeChild(hijo)
  • HTML
  • JavaScript
<section id="lista-eventos"> 
    <article class="tarjeta" id="borrar">Evento a eliminar</article> 
</section> 
const lista = document.querySelector("#lista-eventos"); 
const tarjeta = document.querySelector("#borrar");
lista.removeChild(tarjeta);
en este ejemplo

Se elimina la tarjeta indicando al contenedor (lista) cuál es el hijo exacto que debe borrar.
