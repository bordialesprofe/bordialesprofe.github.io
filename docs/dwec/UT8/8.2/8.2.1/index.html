8.2.1 Qué es un evento y addEventListener

Un evento es cualquier acción que ocurre en la página y que puede ser detectada por JavaScript: un clic, una pulsación de tecla, el envío de un formulario o la carga de la página.

Para reaccionar ante un evento se usa **addEventListener()**, que permite indicar:

  • Qué evento se quiere escuchar
  • Qué función se ejecutará cuando ocurra

1️⃣ Escuchar un evento con addEventListener()​

Para asociar una acción a un elemento se utiliza el método addEventListener().

Sintaxis
elemento.addEventListener("tipoEvento", funcion)
  • HTML
  • JavaScript
<button id="btn-saludo">Saludar</button>
const boton = document.querySelector("#btn-saludo");

boton.addEventListener("click", () => {
  alert("Hola");
});
en este ejemplo

Cuando el usuario hace clic en el botón, se ejecuta la función asociada y se muestra un mensaje.

2️⃣ Separar la función del listener​

Es recomendable definir la función aparte cuando el código es más complejo. Esto mejora la claridad y permite reutilizar la función.

Sintaxis
elemento.addEventListener("tipoEvento", nombreFuncion)
  • HTML
  • JavaScript
<button id="btn-color">Cambiar color</button>
const boton = document.querySelector("#btn-color");

function cambiarColor() {
  boton.classList.toggle("activo");
}

boton.addEventListener("click", cambiarColor);
en este ejemplo

Se define una función llamada cambiarColor y se pasa como referencia al evento click. Cuando se produce el clic, se ejecuta esa función.

3️⃣ No usar atributos HTML como onclick​

Aunque es posible escribir eventos directamente en HTML (onclick="..."), no es recomendable en aplicaciones modernas.

onclik en HTML
<button onclick="alert('Hola')">Saludar</button>

Esta forma mezcla HTML y JavaScript, dificulta el mantenimiento y rompe la separación entre estructura y comportamiento.

Buena práctica

En aplicaciones bien estructuradas se recomienda usar siempre addEventListener() desde JavaScript en lugar de atributos como onclick en el HTML. Esto mantiene separadas la estructura (HTML) y el comportamiento (JavaScript) y facilita el mantenimiento del código.
